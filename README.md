# About

MCU based Standalone SDR implementation with LCD using multicore and integrated high speed ADC.

This project is for article of Interface magazine (Aug 2016) from CQ publishing.


# Prerequisite

* Hardware
   * LPC-Link2 
   * I2S Codec & LCD daughterboard
* IDE
   * LPCXpresso
